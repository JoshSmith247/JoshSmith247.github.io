"Adventure Game" is a web game that is currently in early development. In the end, the game aims to have multiple randomly generated maps with paired campaign setups for an intelligent story mode. This would also include singleplayer and multiplayer online rooms with live chat and trading. Check on this page for updates and working development previews of the game. Contact the developers on discord to ask about the game. Opportunities to contribute assets/artwork will become available sometime in the future.
